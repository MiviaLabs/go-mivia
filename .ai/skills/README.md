# Local Skills

These skills are repo-local workflow contracts. They do not replace global agent skills; they define how generic agents should work inside this repository.

Available skills:

- `project-plan`
- `project-implement`
- `project-review`
- `security-review`
- `mivialabs-agent-mcp`

Usage rules:

- Read `.ai/INDEX.md` before using any local skill.
- Keep work phase-scoped.
- Do not copy repository-specific rules from other projects.
- Write durable handoffs for any incomplete or multi-phase work.
