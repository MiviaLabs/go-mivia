# Active Tasks

Store active task plans and in-progress phase handoffs here.

File guidance:

- One task per Markdown file.
- Include status, owner assumptions, last verified date, phase scope, verification commands, and next-agent prompt.
- Keep claims grounded in current repository state.
- Move completed summaries to `.ai/tasks/done/`.

Do not store:

- Secrets.
- Raw prompts.
- Raw fetched source content.
- PII or personal data.
- Provider tokens or credentials.
